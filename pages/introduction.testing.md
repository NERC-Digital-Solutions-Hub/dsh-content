# ![stars icon](https://cdn.jsdelivr.net/npm/bootstrap-icons@1.13.1/icons/stars.svg?variant=accent) Welcome to the NERC DSH User Testing Site

Thank you for participating in the user testing of the **NERC Digital Solutions Hub (DSH) Data Catalogue** and the **UPRN Service**.  
This site has been created to help us gather feedback, understand your experience, and improve these services for the broader community.

---

#### ![info-circle icon](https://cdn.jsdelivr.net/npm/bootstrap-icons@1.13.1/icons/info-circle.svg) What this is

This is a **user testing site** for:

![search icon](https://cdn.jsdelivr.net/npm/bootstrap-icons@1.13.1/icons/search.svg?variant=primary) **Data Catalogue** – navigating large volumes of documentation can be a significant challenge. To address this, we've developed an AI-powered system that processes and analyses a collection of publicly available support documents, making the information within them easier to access and understand.

![geo-alt icon](https://cdn.jsdelivr.net/npm/bootstrap-icons@1.13.1/icons/geo-alt.svg?variant=primary) **UPRN Service** – this service allows users to extract a wide range of environmental datasets attached to Unique Property Reference Numbers (UPRNs). The primary purpose of this service is to download UPRNs tagged with environmental indicators. You then import this data into your own software for further analysis.

This testing environment allows you to explore functionality and features ahead of the official launch.
